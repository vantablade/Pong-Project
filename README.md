# Pong Chrome Extension 

A simple Pong game built as a Chrome extension for a school project.
Play directly in your browser with quick, responsive controls and classic arcade-style gameplay.

## Features

* Classic Pong gameplay
* Runs as a Chrome extension (no separate app needed)
* Lightweight and fast
* Simple controls and clean design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Install

1. Download or clone this repository
2. Open Google Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked**
5. Select this project folder

## How to Play

* Click the extension icon in your Chrome toolbar
* Control the paddle using your keyboard (e.g. arrow keys or W/S)
* Try to keep the ball in play and beat your high score

## Purpose

This project was created to learn the basics of:

* Game development
* JavaScript interaction and logic
* Building and running Chrome extensions
